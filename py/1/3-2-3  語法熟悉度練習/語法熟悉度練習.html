說明:

1. 在 Python IDLE編輯器中可以從每一行程式的顏色看出程式是否正確。

   (內定顏色)

   註解:              紅色

   字串、多行註解:    綠色

   迴圈 for、if:      棕色

   輸出 input、print: 粉紅色

2. 以 tab 鍵進行內縮，不要使用空白內縮。要對齊才算是同區塊程式碼，若用 4 個空白內縮，則同區塊程式都要用 4 個空白對齊。

   if True:

       print "Yes"

   else:

       print "No"

3. 使用分號 ; 將多個命令放在同一行。

   a=4 ; b=3 ; print (a+b)

4. # (用#開頭) 就是單行註解。

5. '''  為多行註解，即是連續三個單引號。
